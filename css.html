Formulários Tabela

Um Formulário Tabela deve ser utilizado para a exibição e edição de dados tabulares. Existe um estilo padrão definido para todos os formulários tabela quando utilizando a classe .syo-form-table.

Base

Estilo base de todos os formulários tabela.

Pequeno

Permite diminuir as margens e tamanho de fonte para viabilizar a exibição de uma quantidade maior de informação.
Para isto basta adicionar o class syo-small.
